============================================
TG AI 캠퍼스 - 다운로드 및 설치 가이드
============================================

📥 다운로드 방법:
------------------
1. GenSpark 화면 상단의 "Publish" 탭 클릭
2. "Download ZIP" 버튼 클릭
3. 다운로드된 ZIP 파일을 원하는 위치에 압축 해제


💻 로컬 PC에서 실행:
------------------
1. 압축 해제한 폴더 열기
2. "index.html" 파일을 찾기
3. index.html 파일을 더블클릭 (또는 우클릭 → Chrome으로 열기)
4. 브라우저에서 사이트가 열립니다!


📁 필수 폴더 구조:
------------------
프로젝트폴더/
├── index.html          ← 메인 페이지 (이 파일을 여세요!)
├── dashboard.html
├── courses.html
├── about.html
├── programs.html
├── manuals.html
├── dictionary.html
├── support.html
├── notices.html
├── inquiry.html
├── recommended.html
├── css/               ← 스타일 파일 폴더
│   ├── style.css
│   ├── samsung-nav.css
│   └── ...
├── images/            ← 이미지 파일 폴더
│   ├── hero-banner.jpg
│   └── ...
├── js/                ← 자바스크립트 폴더
└── site2/             ← 서브 사이트 폴더


🌐 카페24 호스팅 업로드:
------------------
1. FileZilla 설치 (무료 FTP 프로그램)
   다운로드: https://filezilla-project.org/

2. FileZilla 실행 후 FTP 접속:
   - 호스트: ftp.귀하의도메인.com
   - 사용자명: FTP 아이디
   - 비밀번호: FTP 비밀번호
   - 포트: 21

3. 파일 업로드:
   - 왼쪽: 압축 해제한 로컬 폴더
   - 오른쪽: 서버의 /www 또는 /public_html 폴더
   - 모든 파일과 폴더를 드래그하여 업로드

4. 업로드 완료 후:
   - http://귀하의도메인.com 접속
   - 사이트가 정상 작동됩니다!


⚠️ 주의사항:
------------------
- 모든 폴더(css, images, js, site2)를 함께 업로드해야 합니다
- 폴더 구조를 그대로 유지해주세요
- index.html이 루트 폴더에 있어야 합니다


🆘 문제 해결:
------------------
Q: 디자인이 깨져 보여요
A: css 폴더가 index.html과 같은 위치에 있는지 확인하세요

Q: 이미지가 안 보여요
A: images 폴더가 함께 업로드되었는지 확인하세요

Q: 메뉴가 작동 안 해요
A: 로컬에서는 일부 링크가 작동 안 될 수 있습니다
   서버에 업로드하면 정상 작동됩니다
