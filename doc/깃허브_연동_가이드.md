## ✅ 1단계: 깃허브 저장소 생성 (완료)

### 🔧 이미 완료된 작업:
- ✅ Git 저장소 초기화

## 🎯 2단계: 깃허브 저장소 생성

### 🔗 깃허브에서 직접 생성:
1. [github.com](https://github.com) 접속
2. 로그인 후 우측 상단 `+` 버튼 클릭
3. `New repository` 선택
4. 저장소 정보 입력:
   - **Repository name**: `TG-AI-Campus`
   - **Description**: `전사 직원 AI 교육을 위한 대형 프로젝트 수준 LMS 플랫폼`
   - **Visibility**: `Private` (내부 공유용) 또는 `Public` (공개용)
   - **README**: `Add a README file` ❌ (체크 해제)
   - **.gitignore**: `None` ❌ (체크 해제)
   - **License**: `None` ❌ (체크 해제)

## 🔄 3단계: 원격 저장소 연결

### 🔑 깃허브에서 명령어 복사:
저장소 생성 후 나타나는 명령어 중 아래 부분 복사:

```bash
# 기존 저장소에 원격 추가
git remote add origin https://github.com/[사용자명]/TG-AI-Campus.git
git branch -M main
git push -u origin main
```

## 🌐 4단계: GitHub Pages로 배포

### 📄 배포 설정:
1. 깃허브 저장소에서 `Settings` 탭 클릭
2. 왼쪽 메뉴에서 `Pages` 선택
3. `Source`에서 `Deploy from a branch` 선택
4. `Branch`를 `main`으로, `Folder`를 `/root`로 설정
5. `Save` 클릭

### 🔗 사이트 접속:
- 배포 완료 후: `https://[사용자명].github.io/TG-AI-Campus`
- 약 1-5분 후 접속 가능

## 👥 5단계: 내부 직원 공유 방법

### 🔒 Private 저장소 공유:
1. 깃허브 저장소에서 `Settings` → `Collaborators`
2. `Add people` 클릭
3. 직원들의 깃허브 아이디 또는 이메일 초대
4. 권한 설정: `Read` (읽기 전용) 또는 `Maintain` (전체 권한)

### 📧 공유할 링크:
