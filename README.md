## 🎮 주요 기능 사용 방법

### 1. 강의 시청 및 진도 관리
```
1. courses.html 접속
2. 플레이리스트에서 강의 선택
3. 비디오 시청 → 자동으로 진도 저장
4. 페이지를 나갔다 돌아와도 진도 유지
5. "이어보기" 제안 (10초 이상 시청 시)
```

### 2. 노트 작성
```
1. 강의 시청 중 "노트 작성" 버튼 클릭
2. 현재 재생 시점이 타임스탬프로 자동 기록
3. 메모 작성 후 저장
4. "내 노트" 탭에서 확인
5. 노트의 타임스탬프 클릭 → 해당 시점으로 이동
```

### 3. 퀴즈 풀기
```
1. 강의 시청 완료 후 "퀴즈 풀기" 클릭
2. 카테고리별 맞춤 문제 출제 (3문제)
3. 답안 선택 후 제출
4. 즉시 채점 및 결과 확인
5. 70% 이상 합격 / 미만 불합격
6. 대시보드에서 성적 확인 가능
```

### 4. 북마크 및 리뷰
```
1. 관심 있는 강의에 북마크 추가
2. 대시보드에서 북마크 목록 확인
3. 강의 완료 후 별점 및 리뷰 작성
4. 평균 평점이 강의 정보에 표시
```

### 5. 대시보드 확인
```
1. dashboard.html 접속
2. 학습 통계 한눈에 확인
3. 진행 중인 강의 이어보기
4. 북마크, 노트, 퀴즈 성적 관리
```

## 💾 데이터 저장 구조 (LocalStorage)

### 저장되는 데이터
```javascript
// 1. 학습 진도
ai_learning_progress: {
  "1": { watched: 125, duration: 930, completed: false, lastWatched: "2026-03-20T..." },
  "2": { watched: 1120, duration: 1125, completed: true, lastWatched: "..." }
}

// 2. 북마크
ai_learning_bookmarks: [1, 3, 5, 7]

// 3. 노트
ai_learning_notes: {
  "1": [
    { id: 1711234567890, timestamp: 125.5, text: "중요한 내용...", createdAt: "..." }
  ]
}

// 4. 리뷰
ai_learning_reviews: {
  "1": [
    { id: 1711234567890, rating: 5, text: "훌륭한 강의!", createdAt: "..." }
  ]
}

// 5. 퀴즈 결과
ai_learning_quiz_results: {
  "1": [
    { id: ..., score: 3, totalQuestions: 3, percentage: 100, answers: [...], completedAt: "..." }
  ]
}

// 6. 시청 시간
ai_learning_watch_time: {
  "1": 3600,  // 초 단위
  "2": 1800
}

// 7. 설정
ai_learning_settings: {
  darkMode: false,
  playbackSpeed: 1.0,
  autoplay: true
}
```

## ⚡ 구현된 고급 기능 상세

### 1. 진도 추적 시스템
- **자동 저장**: 5초마다 현재 시청 위치 저장
- **완료 판정**: 90% 이상 시청 시 자동 완료 처리
- **이어보기**: 10초 이상 시청 후 재방문 시 이어보기 제안
- **시청 시간**: 실제 시청 시간 누적 (빨리 감기 제외)

### 2. 노트 시스템
- **타임스탬프**: 작성 시점의 비디오 시간 자동 기록
- **빠른 이동**: 노트 클릭으로 해당 시점 재생
- **실시간 동기화**: 작성 즉시 "내 노트" 탭에 표시
- **삭제 기능**: 불필요한 노트 제거

### 3. 퀴즈 엔진
- **동적 생성**: 카테고리별 문제 은행에서 출제
- **즉시 채점**: 제출 즉시 점수 계산
- **상세 결과**: 정답/오답 표시, 정답률 계산
- **재도전**: 무제한 재도전 가능
- **히스토리**: 모든 시도 기록 저장

### 4. 차트 시각화
- **Chart.js 활용**: 아름다운 인터랙티브 차트
- **실시간 반영**: 학습 데이터 변경 즉시 업데이트
- **반응형**: 모바일/데스크톱 최적화

### 5. 다크모드
- **전역 테마**: 모든 페이지 일관된 테마
- **설정 저장**: 페이지 새로고침해도 유지
- **부드러운 전환**: CSS transition 애니메이션

## ⚠️ 알려진 제한사항

### 1. 데이터 지속성
- **브라우저별 저장**: 다른 브라우저에서 접속 시 데이터 없음
- **기기별 저장**: 다른 기기에서 접속 시 데이터 없음
- **LocalStorage 한계**: 5-10MB 저장 제한
- **영구 삭제 시**: 브라우저 데이터 삭제 시 모든 진도 소실

### 2. 기능 제한
- ❌ 실제 사용자 계정 시스템
- ❌ 서버 간 데이터 동기화
- ❌ 실시간 협업 기능
- ❌ 파일 업로드 (서버 저장)
- ❌ 결제 시스템

### 3. 해결 방법
```
→ Firebase 연동: 실제 백엔드 없이 사용자 인증 및 데이터 동기화
→ Supabase: 오픈소스 Firebase 대안
→ IndexedDB: 더 큰 저장 공간 필요 시
→ 서버 구축: 본격적인 서비스 시
```
