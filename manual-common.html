1. 이메일 작성 자동화

📧 업무 이메일 초안 작성

🎯 실전 프롬프트: 
**상황:** 프로젝트 일정 변경을 알리는 이메일

**받는 사람:** 팀원 5명

**주요 내용:**
- 기존 마감일: 4월 15일
- 변경된 마감일: 4월 25일 (10일 연장)
- 변경 사유: 클라이언트 추가 요구사항 반영
- 수정된 일정표 첨부 예정

**요청:**
정중하고 명확한 업무 이메일 작성

**톤:** 전문적이면서 협조적인
**길이:** 200자 이내
제목: [중요] 프로젝트 일정 변경 안내

팀원 여러분,

클라이언트의 추가 요구사항 반영을 위해 프로젝트 마감일을 조정하게 되었습니다.

변경 일정:
• 기존: 4월 15일 → 변경: 4월 25일 (10일 연장)

수정된 세부 일정표는 별도 첨부드리니 확인 부탁드립니다.
추가 논의 필요 시 언제든 말씀해주세요.

감사합니다.

📬 고객 응대 이메일

🎯 실전 프롬프트: 
**상황:** 고객 문의 답변 이메일

**문의 내용:** "제품 사용 중 오류가 발생했습니다."

**답변 방향:**
1. 문의에 대한 감사 표현
2. 불편 사과
3. 해결 방법 3가지 안내
   - 앱 재시작
   - 캐시 삭제
   - 최신 버전 업데이트
4. 추가 지원 안내

**톤:** 친절하고 해결 지향적
**길이:** 250자 이내
OO님, 안녕하세요. TG 고객지원팀입니다.

소중한 의견 주셔서 감사드리며, 불편을 드려 죄송합니다.

해결 방법:
1️⃣ 앱을 완전히 종료 후 재시작
2️⃣ 설정 > 캐시 삭제
3️⃣ 앱스토어에서 최신 버전(v2.1.5)으로 업데이트

위 방법으로도 해결되지 않으면 support@tg.com 또는 1:1 문의로 연락 주시면 즉시 도와드리겠습니다.

감사합니다.
이메일 자동화 팁: 자주 보내는 이메일 유형별로 템플릿을 만들어두고, AI에게 "상황에 맞게 수정"만 요청하면 시간을 70% 절약할 수 있습니다.

2. 보고서 정리 자동화

📊 회의 내용 보고서 작성

🎯 실전 프롬프트: 
**회의 메모 (자유 형식):**
- 신제품 런칭 일정 논의
- A안 vs B안 비교, 결국 A안 선택
- 예산 3000만원 확정
- 마케팅팀이랑 협업 필요
- 다음주 목요일까지 기획서 제출

**요청:**
위 메모를 정리된 회의록 형식으로 작성

**구성:**
1. 회의 안건
2. 주요 논의 사항
3. 결정 사항
4. 향후 일정
5. 담당자별 액션 아이템

**형식:** 표 또는 리스트
📋 회의록

1. 회의 안건
• 신제품 런칭 전략 수립

2. 주요 논의 사항
• 런칭 전략 A안 vs B안 비교 검토
• 예산 편성 및 마케팅 협업 방안

3. 결정 사항
• 최종 전략: A안 채택
• 총 예산: 3,000만원
• 마케팅팀과 공동 프로젝트 진행

4. 향후 일정
• 4월 10일(목): 상세 기획서 제출

5. 액션 아이템
| 담당자 | 업무 | 마감일 |
|--------|------|--------|
| 김OO | 기획서 작성 | 4/10 |
| 마케팅팀 | 협업 일정 조율 | 4/5 |

📈 데이터 요약 보고서

🎯 실전 프롬프트: 
**원본 데이터:**
- 3월 매출: 5억 2천만원 (전월 대비 +12%)
- 신규 고객: 1,250명 (전월 대비 +8%)
- 재구매율: 38% (전월 대비 -2%p)
- 고객 만족도: 4.3/5.0 (전월 대비 +0.2)

**요청:**
경영진 보고용 요약 보고서 작성

**구성:**
1. 핵심 성과 요약 (3줄)
2. 긍정적 지표 분석
3. 개선 필요 지표 및 대응 방안
4. 다음달 목표

**톤:** 간결하고 데이터 중심
📊 3월 실적 요약 보고

핵심 성과
• 매출 5.2억원 달성 (전월 대비 12% ↑)
• 신규 고객 1,250명 유입 (8% ↑)
• 고객 만족도 4.3/5.0 기록 (0.2p ↑)

✅ 주요 성과
• 매출 성장세 지속 (3개월 연속 10% 이상 성장)
• 신규 고객 유입 안정적
• 만족도 개선 노력 결실

⚠️ 개선 필요
• 재구매율 38% (전월 대비 -2%p)
→ 대응: 고객 리텐션 캠페인 강화 (4월 시행)

🎯 4월 목표
• 매출 5.5억원 (+6%)
• 재구매율 42% 회복

3. 일정 관리 자동화

📅 주간 일정 정리

🎯 실전 프롬프트: 
**산발적인 일정 정보:**
- 월요일 오전 10시 A팀 회의
- 화요일 오후 3시 클라이언트 미팅 (강남)
- 수요일 종일 워크숍 (제주)
- 목요일 오전 기획서 제출
- 금요일 오후 2시 발표

**요청:**
보기 쉬운 주간 일정표 작성

**형식:**
- 날짜별로 정리
- 시간 순서대로
- 중요도 표시 (🔴 긴급, 🟡 중요, 🟢 일반)
- 준비물/체크사항 포함
📅 이번 주 일정 (4월 7일~11일)

월요일 (4/7)
🟡 10:00 | A팀 정기 회의 (회의실 3)
✔️ 준비: 지난주 실적 보고서

화요일 (4/8)
🔴 15:00 | 클라이언트 미팅 (강남 본사)
✔️ 준비: 제안서, 명함, 노트북
✔️ 이동 시간 고려 (14:00 출발)

수요일 (4/9)
🟡 09:00~18:00 | 전사 워크숍 (제주)
✔️ 전날 출장

목요일 (4/10)
🔴 마감 | 신제품 기획서 제출
✔️ 오전 중 최종 검토

금요일 (4/11)
🟡 14:00 | 기획안 발표 (대회의실)
✔️ 준비: PPT, 리허설
캘린더 앱과 AI를 함께 사용하면 더 강력합니다. AI로 일정을 정리한 후 Google Calendar나 Outlook에 한 번에 입력하면 일정 관리가 훨씬 편해집니다.

4. 태스크 관리 자동화

✅ 할 일 우선순위 정리

🎯 실전 프롬프트: 
**오늘 해야 할 일 (무작위 순서):**
- 이메일 50통 답장
- 기획서 수정 (마감 내일)
- 팀 회의 참석 (오전 10시)
- 클라이언트 전화 (긴급)
- 보고서 초안 작성
- 점심 회식 참석
- 데이터 정리

**요청:**
우선순위 기준으로 재정리

**기준:**
1. 긴급도 (마감, 시간 제약)
2. 중요도 (업무 영향도)
3. 소요 시간

**출력:** 번호 순서대로 처리 순서
📋 오늘의 우선순위 태스크

🔴 긴급 (바로 처리)
1. 클라이언트 전화 (긴급) - 10분
2. 팀 회의 참석 (10:00) - 1시간

🟡 중요 (오늘 중 완료)
3. 기획서 수정 (내일 마감) - 2시간
4. 보고서 초안 작성 - 1.5시간

🟢 일반 (가능하면 처리)
5. 이메일 답장 (긴급 건만) - 30분
6. 데이터 정리 - 1시간

🍽️ 고정 일정
• 점심 회식 (12:00)

💡 추천 일정:
09:00 클라이언트 전화
10:00 팀 회의
11:00 기획서 수정 시작
12:00 점심 회식
14:00 기획서 마무리
15:30 보고서 작성
17:00 이메일 처리

5. 회의록 작성 자동화

📝 회의 메모 → 정식 회의록

🎯 실전 프롬프트: 
**회의 음성 녹음 텍스트 (AI 자동 변환):**
"오늘은 마케팅 캠페인에 대해 얘기할게요. 
김대리가 A안을 제안했는데 예산이 좀 많이 들어요. 
박과장은 B안을 제안했는데 더 실용적인 것 같아요. 
결국 B안으로 가기로 했고요, 
다음주 월요일까지 상세 기획서를 만들기로 했습니다."

**요청:**
이 내용을 공식 회의록으로 작성

**포함 사항:**
- 회의 주제
- 참석자
- 논의 내용 (A안/B안 비교)
- 결정 사항
- 후속 조치 및 담당자

**톤:** 공식적이고 명확한
📋 회의록

회의 정보
• 주제: 마케팅 캠페인 전략 수립
• 참석자: 김대리, 박과장

논의 내용
1. 김대리 제안 (A안)
- 장점: 효과적
- 단점: 예산 초과 우려

2. 박과장 제안 (B안)
- 장점: 실용적, 예산 내 진행 가능
- 단점: (논의되지 않음)

결정 사항
• B안으로 최종 확정
• 사유: 실용성 및 예산 효율성

후속 조치
| 담당자 | 업무 내용 | 마감일 |
|--------|-----------|--------|
| 박과장 | B안 상세 기획서 작성 | 4/14(월) |

다음 회의
• 일시: 4월 14일 검토 후 결정
회의 중 AI 음성 변환 앱(Otter.ai, Clova Note 등)을 사용하면 실시간으로 텍스트 변환되어 회의록 작성이 5분 안에 끝납니다!
