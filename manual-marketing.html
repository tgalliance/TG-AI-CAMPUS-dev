마케팅·홍보팀 AI 활용 매뉴얼

읽는 시간: 40분 대상: 마케터/홍보 담당자 실전 프롬프트 25+

1. 카피라이팅 AI 활용법

✍️ 제품 소개 카피 작성

🎯 실전 프롬프트: 
당신은 10년 경력의 카피라이터입니다.

**제품 정보:**
- 제품명: 스마트 워치 S7
- 타겟: 30~40대 직장인
- USP: 7일 배터리, AI 건강 분석, 50m 방수
- 가격: 299,000원

**요청:**
제품 상세페이지 카피 작성

**구성:**
1. 메인 헤드라인 (10자 이내, 임팩트 있게)
2. 서브 헤드라인 (20자 이내, 핵심 기능)
3. 바디 카피 (200자, 감성+논리)
4. CTA 문구 (클릭 유도, 5가지 버전)

**톤:** 프로페셔널하면서 친근함
메인 헤드라인: "일주일, 한 번만 충전하세요"
서브 헤드라인: "AI가 당신의 건강을 24시간 지킵니다"
바디 카피: 아침 조깅부터 밤 수면까지, S7이 함께합니다. 7일 배터리로 주말 여행에도 충전기 걱정 없고, AI 건강 분석은 스트레스, 수면, 심박수를 실시간 모니터링해 당신만의 건강 코치가 되어드립니다...
CTA 예시:
  • "지금 바로 시작하기 →"
  • "한정 수량 확인하기"
  • "무료 배송받고 구매하기"

📣 광고 문구 대량 생성

🎯 실전 프롬프트 (A/B 테스트용): 
**캠페인:** 봄맞이 신학기 프로모션

**제품:** 온라인 강의 플랫폼 구독권

**할인:** 30% 할인 (3월 한정)

**타겟:** 
- 대학생
- 직장인 재교육

**요청:**
네이버 검색광고 제목 20개 생성
- 각 15자 이내
- 긴급성 강조
- 숫자 포함
- 5개는 할인 강조
- 5개는 혜택 강조
- 5개는 후기 강조
- 5개는 호기심 유발
AI가 생성한 광고 문구는 CTR(클릭률)을 높이기 위해 A/B 테스트를 진행하세요. 최소 3~5개 버전을 동시에 운영해 데이터를 모으고, 가장 성과 좋은 버전을 확대하세요.

2. SNS 마케팅 AI 활용법

📸 인스타그램 콘텐츠 기획

🎯 실전 프롬프트: 
당신은 인스타그램 마케팅 전문가입니다.

**브랜드:** 친환경 화장품 브랜드

**계정 톤:** 밝고 친근한, 자연 친화적

**목표:** 팔로워 참여율 UP

**요청:**
이번 주 인스타그램 피드 게시물 5개 기획

**각 게시물마다 포함:**
1. 메인 메시지 (한 줄)
2. 캡션 (150자 이내, 이모지 포함)
3. 해시태그 15개 (인기 + 니치)
4. 이미지 콘셉트 설명
5. 최적 게시 시간 추천

**게시물 타입 분배:**
- 제품 소개: 2개
- 팁/정보: 2개
- 참여 유도(질문/투표): 1개

🎬 숏폼 콘텐츠 스크립트

🎯 실전 프롬프트 (릴스/틱톡): 
**주제:** "직장인 점심시간 5분 스트레칭"

**길이:** 30초

**타겟:** 20~30대 직장인

**목표:** 바이럴, 저장률 UP

**요청:**
릴스 영상 스크립트 작성

**구성:**
- 0~3초: 훅 (시선 끄는 문구)
- 3~25초: 메인 콘텐츠 (3가지 동작)
- 25~30초: CTA (팔로우 유도)

**톤:** 빠르고 에너제틱
**자막:** 큰 글씨로 보이게
**BGM 추천:** 밝고 경쾌한 음악
[0~3초 훅]
"앉아만 있으면 목이 뻐근하죠? 지금 바로 따라해보세요!"

[3~25초 메인]
1️⃣ 목 돌리기 (좌우 10초)
2️⃣ 어깨 펴기 (뒤로 당기기 10초)
3️⃣ 허리 비틀기 (좌우 10초)

[25~30초 CTA]
"매일 점심 후 5분! 저장하고 따라해보세요 ✨"

📊 SNS 콘텐츠 캘린더 작성

🎯 실전 프롬프트: 
**채널:** 인스타그램, 페이스북, 링크드인

**기간:** 2026년 4월 (1개월)

**주요 이벤트:**
- 4월 5일: 신제품 출시
- 4월 20일: 창립기념일 이벤트

**요청:**
월간 SNS 콘텐츠 캘린더 작성

**형식:**
표 형식 (날짜 | 채널 | 콘텐츠 타입 | 주제 | 목표)

**포함 사항:**
- 주 3회 게시 (월/수/금)
- 채널별 특성 반영
- 신제품 론칭 프로모션 일정
- 이벤트 전후 티저/결과 콘텐츠

3. 콘텐츠 제작 AI 활용법

📝 블로그 포스트 작성

🎯 실전 프롬프트: 
당신은 콘텐츠 마케터입니다.

**주제:** "2026년 디지털 마케팅 트렌드 TOP 7"

**목표:** SEO 상위 노출, 공유율 UP

**타겟 키워드:** "디지털 마케팅 트렌드"

**요청:**
블로그 포스트 작성 (2,000자)

**구성:**
1. 서론 (200자): 트렌드 중요성
2. 본론 (1,500자): 7가지 트렌드 상세 설명
   - 각 트렌드마다: 정의 + 사례 + 적용 팁
3. 결론 (300자): 실행 방안 요약

**추가 요청:**
- 각 섹션에 소제목
- 인용문 1~2개
- 통계 데이터 포함
- 내부 링크 제안 3개

🎥 유튜브 대본 작성

🎯 실전 프롬프트: 
**채널:** B2B SaaS 제품 소개

**영상 길이:** 5분

**제목:** "3분 만에 이해하는 우리 서비스"

**타겟:** 중소기업 대표/임원

**요청:**
유튜브 영상 대본 작성

**구성:**
- 인트로 (0~30초): 문제 제기
- 솔루션 소개 (30초~3분): 기능 3가지
- 고객 후기 (3~4분): 짧은 인터뷰
- CTA (4~5분): 무료 체험 유도

**톤:** 전문적이지만 이해하기 쉽게
**주의:** 전문 용어 최소화
유튜브 대본은 AI가 초안을 작성한 후, 실제 녹화하면서 자연스러운 말투로 수정하세요. "대본 읽기" 티가 나지 않도록 구어체로 다듬는 것이 중요합니다.

4. 이메일 마케팅 AI 활용법

📧 뉴스레터 작성

🎯 실전 프롬프트: 
당신은 이메일 마케팅 전문가입니다.

**발송 대상:** 구독자 5,000명 (기존 고객 + 리드)

**주제:** 3월 신제품 출시 + 특별 할인

**목표:** 오픈율 25%, 클릭률 5%

**요청:**
뉴스레터 이메일 작성

**구성:**
1. 제목 (50자 이내, 오픈율 높이기)
2. 프리헤더 텍스트 (30자, 미리보기)
3. 본문 (400자)
   - 신제품 소개
   - 할인 혜택 (30% OFF, 3일 한정)
   - 고객 후기 1개
4. CTA 버튼 문구 3가지

**톤:** 친근하고 긴급함 전달
**주의:** 스팸 필터링 피하기 (과도한 대문자, 느낌표 금지)

🎯 세그먼트별 맞춤 메시지

🎯 실전 프롬프트: 
**상황:**
고객 세그먼트별 이메일 작성

**세그먼트 3가지:**
1. 신규 가입자 (가입 후 7일 이내)
2. 활성 고객 (최근 30일 구매 이력)
3. 휴면 고객 (90일 이상 미구매)

**공통 목표:** 구매 전환

**요청:**
각 세그먼트별 이메일 작성 (각 300자)

**차별화 포인트:**
- 신규: 환영 + 첫 구매 할인
- 활성: 감사 + 추천 제품
- 휴면: 그리움 + 특별 혜택

**각 이메일마다:**
- 제목
- 본문
- CTA
신규 가입자 이메일:
제목: "환영합니다! 첫 구매 20% 할인 쿠폰 🎁"
본문: OO님, TG Store에 가입해주셔서 감사합니다! 새로운 여정을 축하하며 특별 선물을 준비했어요. 지금 바로 사용 가능한 20% 할인 쿠폰을 받아보세요...

휴면 고객 이메일:
제목: "OO님이 그리워요... 돌아오시면 30% 드릴게요 💙"
본문: 오랜만이에요, OO님! 그동안 잘 지내셨나요? 90일 동안 찾아오지 않으셔서 많이 그리웠어요. 돌아오신 당신을 위해 특별한 혜택을 준비했습니다...

5. SEO 최적화 AI 활용법

🔍 키워드 리서치

🎯 실전 프롬프트: 
**주제:** "홈트레이닝" 관련 콘텐츠

**목표:** 네이버/구글 상위 노출

**요청:**
SEO 키워드 전략 수립

**분석 항목:**
1. 메인 키워드 후보 5개
   - 검색량 추정
   - 경쟁 강도 예상
2. 롱테일 키워드 15개
3. 연관 검색어 10개
4. 시즌별 키워드 (봄/여름/가을/겨울)

**결과 형식:**
표 (키워드 | 월간 검색량 예상 | 경쟁도 | 추천도)

📄 메타 태그 작성

🎯 실전 프롬프트: 
**페이지:** 제품 상세 페이지

**제품:** 무선 청소기 V10

**주요 키워드:** "무선청소기 추천"

**요청:**
SEO 최적화된 메타 태그 작성

**필요 항목:**
1. 메타 제목 (60자 이내)
   - 키워드 포함
   - 클릭 유도
2. 메타 설명 (155자 이내)
   - USP 포함
   - CTA 포함
3. OG 태그 (소셜 미디어용)
   - og:title
   - og:description
   - og:image 추천 사양

**참고:**
- 검색 결과에서 눈에 띄도록
- 브랜드명 포함
메타 설명은 검색 결과에서 사용자가 클릭하기 전에 보는 첫 인상입니다. USP(핵심 차별점)와 숫자를 포함하면 클릭률이 높아집니다. 예: "배터리 90분, 흡입력 200W, 5단계 필터 ✨"

6. 캠페인 기획 AI 활용법

🎯 마케팅 캠페인 전략 수립

🎯 실전 프롬프트: 
당신은 마케팅 전략 컨설턴트입니다.

**캠페인 목표:**
신규 고객 1,000명 유치 (3개월)

**제품:** B2B HR 소프트웨어

**타겟:**
- 50~200인 중소기업
- 인사팀장/경영진

**예산:** 5,000만원

**요청:**
마케팅 캠페인 전략서 작성

**포함 사항:**
1. 캠페인 컨셉 (슬로건 포함)
2. 채널 전략
   - 온라인: 구글 광고, 링크드인, SEO
   - 오프라인: 세미나, 박람회
3. 예산 배분 (채널별 %)
4. 콘텐츠 계획 (주제 10개)
5. KPI 설정 (목표 수치)
6. 월별 실행 일정

**형식:** A4 3페이지 분량

📊 경쟁사 분석

🎯 실전 프롬프트: 
**우리 브랜드:** TG 친환경 화장품

**경쟁사 3개:**
- A사: 시장 점유율 1위, 대기업
- B사: 가성비 강조, 온라인 중심
- C사: 프리미엄 포지셔닝

**요청:**
경쟁사 마케팅 전략 분석

**분석 항목:**
1. 각 경쟁사의 강점/약점
2. 주요 마케팅 채널
3. 메시징 전략 (어떤 가치 강조?)
4. 가격 정책
5. 우리의 차별화 포인트 3가지
6. 추천 전략 (경쟁사 대비 우위 확보 방안)

**형식:** SWOT 분석 포함
경쟁사 분석은 AI가 공개 정보를 바탕으로 초안을 작성하지만, 실제로는 직접 경쟁사 웹사이트, SNS, 광고를 모니터링하며 업데이트해야 합니다. AI는 출발점이고, 지속적인 관찰과 업데이트가 핵심입니다.

🚀 마케팅 성과를 10배 높이세요!

AI를 활용하면 콘텐츠 제작 시간이 70% 단축됩니다.
