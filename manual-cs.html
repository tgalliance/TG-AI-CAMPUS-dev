1. 고객 응대 AI 활용법

💬 고객 응대 스크립트 작성

🎯 실전 프롬프트: 
당신은 10년 경력의 CS 전문가입니다.

**상황:**
고객이 제품 배송 지연에 대해 불만을 제기

**고객 유형:** 화가 난 상태

**제품:** 온라인 쇼핑몰 (의류)

**배송 지연 사유:** 택배사 과부하 (명절 시즌)

**요청:**
고객 응대 스크립트 작성 (3단계)

**구성:**
1. 공감 및 사과 (50자)
2. 상황 설명 (100자)
3. 해결책 제시 (100자)
   - 배송 현황 실시간 확인 링크
   - 10% 할인 쿠폰 제공
   - 우선 배송 처리

**톤:** 진심 어린 사과, 공손하고 적극적인 태도
1단계 - 공감/사과:
"고객님, 기다리신 제품이 제때 도착하지 않아 정말 죄송합니다. 고객님의 불편한 심정 충분히 이해합니다."

2단계 - 상황 설명:
"명절 시즌으로 인해 택배사의 물량이 평소보다 3배 증가하면서 일시적인 지연이 발생했습니다. 현재 고객님의 상품은 OO 배송센터에서 출고 대기 중이며, 내일 오전 중 출발 예정입니다."

3단계 - 해결책:
"불편을 드린 점 보상하고자 다음 구매 시 사용 가능한 10% 할인 쿠폰을 즉시 발송드렸습니다. 또한 해당 주문건은 최우선 배송으로 처리하여 내일 중 수령하실 수 있도록 조치하였습니다. 실시간 배송 현황은 [링크]에서 확인 가능합니다."

📞 전화 응대 멘트 생성

🎯 실전 프롬프트: 
**업종:** B2B SaaS 제품 고객센터

**상황별 전화 응대 멘트 작성 (5가지)**

**시나리오:**
1. 전화 수신 (첫 인사)
2. 대기 요청 (정보 확인 중)
3. 해결 완료 (감사 인사)
4. 해결 불가 (에스컬레이션)
5. 전화 종료 (마무리 멘트)

**각 시나리오마다:**
- 기본 멘트 (정중한 버전)
- 간결 멘트 (빠른 응대용)

**톤:** 전문적이고 친절한
**회사명:** TG Solutions
1. 전화 수신:
기본: "안녕하세요, TG Solutions 고객센터입니다. 무엇을 도와드릴까요?"
간결: "TG Solutions입니다. 말씀하세요."

2. 대기 요청:
기본: "고객님, 정확한 정보 확인을 위해 잠시만 기다려 주시겠습니까? 1~2분 소요됩니다."
간결: "잠시만 기다려 주세요. 확인해드리겠습니다."

5. 전화 종료:
기본: "오늘 TG Solutions를 이용해주셔서 감사합니다. 추가로 궁금하신 사항이 있으시면 언제든 연락 주세요. 좋은 하루 되십시오!"
전화 응대는 첫 3초가 중요합니다. 밝고 명확한 톤으로 회사명과 인사를 전달하세요. 고객이 화가 난 상황에서는 "~드리겠습니다" 표현보다 "~해드리겠습니다"가 더 진심 어린 느낌을 줍니다.

FAQ 30개 생성 (카테고리별)

4. 기술 지원 (5개)
5. 기타 (5개)

**각 FAQ 구성:**
- 질문 (고객이 실제로 검색할 표현)
- 답변 (200자 이내, 단계별 설명)
- 관련 링크 (도움말 페이지)
- 연관 질문 2개

**톤:** 친근하고 이해하기 쉬운
**추가:** 챗봇에 바로 적용 가능하도록 JSON 형식도 함께 제공

🎯 실전 프롬프트: 
**데이터:**
최근 3개월간 고객센터 이메일 문의 500건

**주요 문의 유형:**
- 배송 관련: 180건
- 제품 불량: 90건
- 환불 요청: 120건
- 사용법 문의: 110건

**요청:**
1. 문의 패턴 분석 (빈도 높은 키워드)
2. 시간대별 문의 분포 예측
3. FAQ 우선순위 10개 추천
4. 자주 묻는 질문 자동 응답 템플릿 5개
5. 고객 불만 예방 방안 3가지

**목표:** 문의량 30% 감소
분석 결과 예시:
• 가장 많은 문의: "배송 조회" (주말 저녁 집중)
• FAQ 우선순위 1위: "주문 후 배송까지 얼마나 걸리나요?"
• 자동 응답 적용 가능: 배송 조회, 반품 절차, 교환 방법
• 예방 방안: ① 주문 확인 이메일에 예상 배송일 명시 ② 실시간 배송 추적 링크 제공 ③ 제품 상세 페이지에 사용 가이드 동영상 추가

3. 영업 제안서 AI 활용법

📋 영업 제안서 작성

🎯 실전 프롬프트: 
당신은 B2B 영업 전문가입니다.

**고객사:** OO 제조업체 (직원 300명)

**고객 니즈:**
- 재고 관리 시스템 도입
- 엑셀 기반 관리에서 벗어나고 싶음
- 예산: 5,000만원 이내

**우리 제품:** 클라우드 기반 스마트 재고 관리 솔루션

**강점:**
- 실시간 재고 파악
- AI 수요 예측
- 모바일 앱 지원
- 도입 후 평균 30% 재고 비용 절감

**요청:**
영업 제안서 작성 (A4 3페이지 분량)

**구성:**
1. 표지 (제목 + 슬로건)
2. 고객사 현황 및 Pain Point (300자)
3. 솔루션 소개 (500자)
4. 주요 기능 (5가지, 각 100자)
5. 도입 효과 (ROI 계산 포함)
6. 견적서 (구성 항목별)
7. 도입 일정 (타임라인)
8. 고객 성공 사례 2개

**톤:** 전문적이고 신뢰감 있는
**강조:** 비용 절감 효과와 ROI
AI가 제안서 초안을 작성하면, 다음을 추가하세요:
• 고객사 로고와 이름을 표지에 넣어 맞춤형 느낌 강조
• 실제 데이터를 기반으로 ROI 계산 (예: 연간 6,000만원 절감 가능)
• 데모 일정 제안 (예: "2주 무료 체험 후 도입 결정")
• 담당자 연락처와 후속 미팅 제안

📊 경쟁 우위 분석표 작성

🎯 실전 프롬프트: 
**우리 제품:** TG 클라우드 CRM

**경쟁사 3개:**
- A사: 국내 1위, 고가 정책
- B사: 해외 솔루션, 한국어 지원 약함
- C사: 저가형, 기능 제한적

**비교 항목:**
- 가격 (월 구독료)
- 주요 기능 (10가지)
- 사용 편의성
- 고객 지원
- 모바일 앱
- 데이터 보안
- 한국어 지원
- 커스터마이징
- 도입 기간
- 레퍼런스

**요청:**
비교표 작성 (표 형식) + 우리의 차별화 포인트 3가지

**목표:** 영업 미팅에서 바로 사용
경쟁 우위 분석표는 항목별로 ⭐(강점), ✓(보통), ✗(약점) 기호를 사용하면 한눈에 비교가 쉽습니다. 우리 제품의 강점은 굵은 글씨로 강조하세요.

5. 챗봇 시나리오 AI 활용법

🤖 챗봇 대화 시나리오 작성

🎯 실전 프롬프트: 
**챗봇 용도:** 웹사이트 고객 문의 자동 응답

**주요 문의 유형:**
1. 제품 정보
2. 주문/배송 조회
3. 환불/교환
4. 기술 지원

**요청:**
챗봇 시나리오 작성 (대화 흐름도)

**구성:**
1. 인사 메시지 (3가지 버전)
2. 메인 메뉴 (4가지 선택지)
3. 각 메뉴별 하위 시나리오
   - 제품 정보: 카테고리 선택 → 상세 정보
   - 주문 조회: 주문번호 입력 → 배송 현황
   - 환불/교환: 사유 선택 → 절차 안내
   - 기술 지원: 문제 유형 → 해결 방법
4. 해결 불가 시 상담원 연결
5. 종료 메시지

**특징:**
- 이모지 사용
- 버튼 선택 방식
- 최대 3단계 깊이
- 평균 응답 시간 3초 이내
인사 메시지:
"안녕하세요! 👋 TG Store 챗봇입니다. 무엇을 도와드릴까요?"

메인 메뉴:
[제품 정보 🛍️] [주문 조회 📦] [환불/교환 🔄] [기술 지원 🔧]

예시 대화 (주문 조회):
고객: [주문 조회 클릭]
봇: "주문번호를 입력해주세요. (예: #2024-1234)"
고객: #2024-1234
봇: "주문하신 '무선 이어폰'은 현재 배송 중이며, 내일 오전 도착 예정입니다. 📦
[실시간 조회] [다른 문의하기]"
챗봇은 80%의 단순 문의를 자동 처리하는 것이 목표입니다. 복잡한 문의는 빠르게 상담원에게 연결하는 것이 고객 만족도를 높입니다. "상담원 연결" 버튼은 모든 단계에서 항상 표시하세요.

6. CRM 데이터 활용 AI 분석

📊 고객 세그먼트 분석

🎯 실전 프롬프트: 
**데이터:**
고객 10,000명의 CRM 데이터

**주요 필드:**
- 가입일, 최근 구매일, 총 구매 금액, 구매 빈도
- 연령대, 지역, 관심 카테고리

**요청:**
고객 세그먼트 분석 및 전략 수립

**분석 항목:**
1. RFM 분석 (Recency, Frequency, Monetary)
   - 최우수 고객 (상위 10%)
   - 잠재 이탈 고객 (90일 이상 미구매)
   - 신규 고객 (가입 30일 이내)
2. 세그먼트별 특성
3. 각 세그먼트별 마케팅 전략 3가지
4. 예상 전환율 및 매출 증대 효과

**목표:** 재구매율 20% 향상

🎯 이탈 고객 예측 및 방지
